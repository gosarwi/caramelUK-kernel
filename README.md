**CaramelUK** (основной компонент проекта **Caramely OS**) — это независимое монолитное хобби-ядро и операционная система, разрабатываемая с нуля. 

Цель проекта — создание легкой, быстрой и независимой системы с собственными текстовыми и графическими подсистемами, свободной от корпоративного оверхеда и телеметрии.

---

 ✨ Ключевые особенности (Цикл "Green Tea")

* **Архитектура:** Компактное монолитное ядро с собственной логикой управления системой.
* **Графика и Интерфейс:** * Поддержка текстового режима VGA и расширений **VBE (VESA BIOS Extensions)** для графического вывода.
  * Интегрированный шрифт **Roboto** для отрисовки элементов интерфейса и улучшения читаемости текста.
  * Полная встроенная поддержка **кириллицы**.
* **Подсистемы:**
  * Собственная **VFS (Виртуальная файловая система)** для работы с файлами и абстракциями.
  * Зачатки графического окружения (Desktop Environment) со стандартным разрешением экрана **800x600**.
  * Небольшая встроенная игра-лабиринт с поддержкой звуковых сигналов (PC Speaker).
